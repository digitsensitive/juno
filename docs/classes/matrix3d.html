Class Matrix3D

A matrix is a mathematical object composed of a set of numerical quantities arÂ­ranged in a two-dimensional array of rows and columns.

This class holds a 3 x 3 matrix in a two-dimensional array ( n x m ).

[[M00,M01,M0(m-1)]
 [M10,M11,M1(m-1)]
 [M(n-1)0,M(n-1)1,M(n-1)(m-1)]]

The arrangement of the entries is in a row-major order (https://en.wikipedia.org/wiki/Row-_and_column-major_order).
